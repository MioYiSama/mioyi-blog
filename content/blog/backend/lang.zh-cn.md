## TypeScript

- 前端就是JS
- DX
- 异步/并发性能

## JVM (Java/Kotlin)

- 生态、社区、资源
- DX（Spring Boot）

唯一的不足是内存占用，但是瑕不掩瑜

## Go

- cloud native
- cloud infrasturcture (Kubernetes, Docker, Terraform, Prometheus)

## Rust

- 纯粹的性能

## 为什么别的语言不行

1. C#：除了性能之外各方面被JVM暴打。微软和windows的刻板印象没法改。
2. Python：各方面被JS暴打
3. PHP/Ruby：全栈语言，不是纯后端
4. Zig：后端是工程化的，太low level（如手动管理内存）会过分降低开发效率
